This is a storage for custom components, wrappers and misc extensions. 
Directory is not added into the exceptions of CFWheels rewrite, 
so any access attempts through the web will throw 404 error.

Current directory contents:

models/ -- extensions for models, mostly to implement non-cfscript and other utility methods

thirdparty/ -- 3rd-party and open-source components

wrappers/ -- 3rd-party API wrappers
